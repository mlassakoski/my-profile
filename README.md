# mlassakoski profile

## Requirements:

 * Nodejs 4 + installed;
 * Npm or Yarn;

## Installation

  * Clone the project from github
  `https://github.com/mlassakoski/my-profile.git`

#### Download project dependencies

  * `yarn install or yarn install`

## Usage 

* Execute `ng serve or yarn start:local` to run the application.
